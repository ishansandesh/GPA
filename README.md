# 📊 Smart GPA Analyzer

A modern, interactive GPA calculator and academic performance analyzer for university students.  
It allows users to input subjects, select difficulty levels, estimate marks, and calculate GPA with real-time chart visualization.

---

## 🚀 Features

- 🎯 Add unlimited subjects dynamically
- 🧠 Select difficulty level (Easy / Medium / Hard)
- 📊 Auto-predict marks for missing inputs
- 🧮 Accurate GPA calculation (weighted by credits)
- 📈 Interactive bar chart visualization (Chart.js)
- 💡 Smart performance feedback system
- ⚡ Real-time calculation
- 🎨 Modern responsive UI design
